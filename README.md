# igendoc-search-ai

## Instruction

1. Create a VM:

```
python3 -m venv myenv
```

2. Activate the VM:

```
source myenv/bin/activate
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the server:

```
python app/app.py
```

5. Test the server by hitting this GET endpoint: [hello-api](http://127.0.0.1:5000/v1/hello)
